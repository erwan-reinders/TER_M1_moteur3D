Class: Kernel

Kernel(kernel, textureReadName, textureWriteName, width, height)

Kernel Classe shader permettant de générer l'image résultante à la convolution entre l'image d'origine et le kernel passé en paramettre.

Constructor

new Kernel(kernel, textureReadName, textureWriteName, width, height)

Construit le faiseur de rendu permettant de faire la convolution avec le kernel.
Parameters:
Name Type Description
kernel Float32Array Le noyau de convolution. La taille doit correspondre à celle du shader utilisé.
textureReadName string Le nom de la texture d'entrée.
textureWriteName string Le nom de la texture de sortie.
width number la résolution horizontale du rendu en nombre de pixel.
height number la résolution verticale du rendu en nombre de pixel.
